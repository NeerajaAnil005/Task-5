Objective
  Learn how to combine data from two related tables using SQL JOINs.

Tables Used
  Customers
  Orders

JOINs Practiced
  INNER JOIN → shows matching rows
  LEFT JOIN → all customers + matching orders
  RIGHT JOIN → all orders + matching customers
  FULL JOIN (using UNION in MySQL)

Outcome
  Ability to merge data from multiple tables and understand how different JOIN types work.
